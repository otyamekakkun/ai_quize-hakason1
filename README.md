# openai_quiz
chatGPTを利用したクイズアプリです。

(使い方) mac Os対応です

git cloneかzip形式でダウンロードしていただき、コマンドでai_quiz-hakason1に入っていただきます。

flask runとコマンドで叩くと自身のローカルサーバーが起動します。

http://127.0.0.1:5000 と検索するとアプリに入ることができます。
